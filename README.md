## Features

### Dashboard
- **Real-time OHLCV charts** for 5 instruments: BTC/USD, ETH/USD, EUR/USD, XAU/USD, NVDA
- **Volume bars** synchronized with candlestick price chart
- **Liquidity depth chart** with bid/ask volume imbalance
- **KPI bar**: P&L, Win Rate, Sharpe Ratio, Max Drawdown, Open Positions, Signals Today

### AI Signals
- **Real-time trade signal feed** generated by simulated RSI, ATR, VWAP, and Order Flow models
- **Explainable AI (XAI) panel**: every signal includes primary reason, contributing indicator values, source algorithm, confidence score, target price, stop loss, and risk/reward ratio
- **Signal distribution chart**: Long vs Short doughnut breakdown

### Anomaly Detection
- **5 anomaly types**: Volume Spike, Flash Move, Liquidity Gap, Order Block, Regime Shift
- **Severity classification**: High / Medium / Low with color-coded badges
- **Anomaly scatter chart**: magnitude vs time
- **Severity histogram**: aggregated by category

### Blockchain Audit Trail
- **SHA-256-like hash chain**: every AI-driven trade signal produces a verified audit entry
- **Hash chain integrity**: each entry includes `txHash` and `previousHash` — tamper-evident
- **Verified badge** per entry

### Strategy Lab
- **6 tunable parameters**: RSI thresholds, ATR multiplier, VWAP band, confidence filter, position size
- **Radar chart**: live performance score visualization vs benchmark

### API Bridge Monitor
- **5 simulated endpoints**: Market Data Feed, Execution Engine, ML Inference API, Audit Chain Node, Risk Engine
- **Latency simulation**: log-normal jitter model (realistic TCP behaviour)
- **Markov status transitions**: connected → degraded → reconnecting → connected
- **System health score**: weighted 0–100 aggregate

### ML Regime Detector
- **Hidden Markov Model** regime state: Trending / Mean-Reverting / Volatile / Consolidating
- **Transition probabilities** displayed per regime
- **Live badge** in sidebar with color-coded indicator

## Getting Started

### Option A — Open directly in browser (no install)

1. Clone or download the repository
2. Open `src/dashboard/axiom-trading.html` in any modern browser
3. Dashboard starts immediately — all data simulated in-memory

### Option B — Dev server with hot reload

```bash
npm install
npm run dev
# Opens at http://localhost:5173/src/dashboard/axiom-trading.html
```

### Option C — Production build

```bash
npm run build
npm run preview
```

## Technical Decisions

| Decision | Rationale |
|---|---|
| Single-file dashboard HTML | Zero-dependency, instant open in browser, GitHub Pages compatible |
| Chart.js 4.4 | Best balance of bundle size and feature set for financial charts |
| GBM price model | Geometric Brownian Motion is the industry standard for short-term price simulation |
| HMM regime detector | Hidden Markov Models are state-of-the-art for financial regime classification |
| FNV-1a mock hash | Deterministic hash simulation without WebCrypto dependency |
| In-memory state (no localStorage) | No persistence side effects; state always fresh on load |
| TypeScript strict mode | `strict + noUncheckedIndexedAccess + exactOptionalPropertyTypes` — maximum type safety |
